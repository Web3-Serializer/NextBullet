API Documentation
REST API v1
Base URL
Stats
GET /api/stats Global statistics
Response
{ "configs": 5, "wordlists": 3, "runners": 2, "active_runners": 1, "total_hits": 42 }
Configs
GET /api/configs List all configs
Response
[{ "id": "uuid", "name": "My Config", "author": "anon", "created_at": "...", "settings": {...}, "blocks": [...] }]
POST /api/configs Create a new config
Body (JSON)
ParamTypeDescription
namestringConfig name
authorstringAuthor name
settingsobject{ max_threads, timeout, data_separator, use_proxies, verbose }
blocksarrayArray of block objects
Response (201)
{ "id": "generated-uuid", "name": "My Config", ... }
GET /api/configs/:id Get a config by ID
Response
{ "id": "...", "name": "...", "blocks": [...], "settings": {...} }
PUT /api/configs/:id Update a config
Body (JSON)
{ "name": "...", "author": "...", "settings": {...}, "blocks": [...] }
DELETE /api/configs/:id Delete a config
Response
{ "ok": true }
Wordlists
GET /api/wordlists List all wordlists
Response
[{ "name": "combos.txt", "size": 1024, "lines": 500, "modified": "..." }]
POST /api/wordlists Upload a wordlist file
Body

Multipart form data with field file

Example (curl)
curl -X POST -F "file=@combos.txt" http://localhost:5000/api/wordlists
DELETE /api/wordlists/:name Delete a wordlist
Response
{ "ok": true }
Runners
GET /api/runners List all runners
Response
[{ "id": "uuid", "name": "Runner abc12345", "config_name": "...", "wordlist": "combos.txt", "status": "running", "total": 1000, "tested": 450, "hits": 12, "fails": 430, "retries": 5, "errors": 3, "bans": 0, "cpm": 320, "progress": 45.0, "threads": 10, "started_at": "...", "finished_at": null }]
POST /api/runners Create and start a new runner
Body (JSON)
ParamTypeDescription
config_idstringConfig UUID (required)
wordliststringWordlist filename (required)
threadsintNumber of threads (default: 10)
proxiesstringProxy list, one per line (optional)
Example
{ "config_id": "a0148c06-ad1d-4531-9994-98b0e855499f", "wordlist": "combos.txt", "threads": 50, "proxies": "http://proxy1:8080\nhttp://proxy2:8080" }
Response (201)
{ "id": "runner-uuid" }
GET /api/runners/:id Get runner status and stats
Response
{ "id": "...", "status": "running", "tested": 450, "total": 1000, "hits": 12, "cpm": 320, ... }
Status values

idle running paused stopped finished error

POST /api/runners/:id/pause Pause a running runner
Response
{ "ok": true }
POST /api/runners/:id/resume Resume a paused runner
Response
{ "ok": true }
POST /api/runners/:id/stop Stop a runner
Response
{ "ok": true }
DELETE /api/runners/:id Remove a runner (stops it first)
Response
{ "ok": true }
GET /api/runners/:id/hits Get hits for a specific runner
Response
[{ "combo": "user:pass", "user": "user", "pass": "pass", "time": "14:30:01", "captures": { "token": "abc" }, "extra": {...} }]
GET /api/runners/:id/log Get live log entries (polling)
Query params
ParamTypeDescription
afterintReturn entries with id > after (for polling)
Response
[{ "id": 42, "time": "14:30:01", "combo": "user:pass", "user": "user", "result": "SUCCESS", "status": "200", "captures": { "token": "abc" }, "extra": {...} }]
Hits Database
GET /api/hits List hits from database
Query params
ParamTypeDescription
searchstringSearch in combo/username/extra
configstringFilter by config name
limitintMax results (default: 500)
offsetintPagination offset
Response
{ "hits": [{ "id": 1, "combo": "...", "username": "...", "password": "...", "config_name": "...", "extra": "{...}", "created_at": "..." }], "total": 150 }
GET /api/hits/export Export hits as text
Query params
ParamTypeDescription
configstringFilter by config name
formatstringcombo | user_pass | capture | full
Response (text/plain)
combo: user:pass capture: user:pass | token=abc123 | email=test@mail.com full: user:pass | token=abc123 | status=200
DELETE /api/hits/:id Delete a single hit
Response
{ "ok": true }
POST /api/hits/clear Clear all hits (or by config)
Body (JSON, optional)
{ "config": "My Config" }
Debugger
POST /api/debugger/step Execute a single pipeline step
Body (JSON)
ParamTypeDescription
blocksarrayArray of block objects
stepintCurrent step index
contextobjectCurrent execution context (variables)
Response
{ "done": false, "step": 0, "block_type": "request", "ok": true, "error": null, "condition": null, "context": { "user": "test", "status": "200", "response": "...", "_result": "NONE" }, "next_step": 1 }
Converter
POST /api/convert Convert LoliScript/OpenBullet config to NextBullet blocks
Body (JSON)
ParamTypeDescription
sourcestringConfig source code
formatstringloliscript | openbullet2
Response
{ "blocks": [{ "type": "request", "method": "POST", "url": "...", ... }] }
